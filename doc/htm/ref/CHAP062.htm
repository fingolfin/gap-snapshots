62 Finitely Presented Algebras

Currently the GAP library contains only few functions dealing with general finitely presented algebras, so this file is merely a placeholder.

The special case of finitely presented Lie algebras is described in Finitely Presented Lie Algebras, and there is also a GAP package fplsa for computing structure constants of finitely presented Lie (super)algebras.
