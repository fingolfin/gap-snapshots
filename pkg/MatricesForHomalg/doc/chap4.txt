  
  [1X4 Ring Maps[0X
  
  A  [5Xhomalg[0m  ring  map is a data structure for maps between finitely generated
  rings.  [5Xhomalg[0m more or less provides the basic declarations and installs the
  generic  methods for ring maps, but it is up to other high level packages to
  install  methods  applicable  to  specific  rings.  For example, the package
  [5XSheaves[0m provides methods for ring maps of (finitely generated) affine rings.

  [1X4.2 Ring Maps: Constructors[0X
  
  [1X4.2-1 RingMap[0m
  
  [2X> RingMap( [0X[3Ximages, S, T[0X[2X ) _________________________________________[0Xoperation
  [6XReturns:[0X  a [5Xhomalg[0m ring map
  
  This  constructor  returns  a  ring map (homomorphism) of finitely generated
  rings/algebras.  It  is  represented  by  the  images  [3Ximages[0m  of the set of
  generators  of  the  source  [5Xhomalg[0m ring [3XS[0m in terms of the generators of the
  target  ring  [3XT[0m  (--> [14X3.2[0m). Unless the source ring is free [13Xand[0m given on free
  ring/algebra  generators the returned map will cautiously be indicated using
  parenthesis:  "homomorphism".  To  verify  if  the  result  is indeed a well
  defined  map  use  [2XIsMorphism[0m  ([14X4.3-1[0m).  If  source and target are identical
  objects, and only then, the ring map is created as a selfmap.
