  [1X3.2 Rings: Constructors[0X
  
  This   section   describes   how   to   construct   rings   for   use   with
  [5XMatricesForHomalg[0m,  which exploit the [5XGAP4[0m-built-in abilities to perform the
  necessary  ring operations. By this we also mean necessary matrix operations
  over  such  rings.  For  the  purposes of [5XMatricesForHomalg[0m only the ring of
  integers  is  properly  supported in [5XGAP4[0m. The [5XGAP4[0m extension packages [5XGauss[0m
  and [5XGaussForHomalg[0m extend these built-in abilities to operations with sparse
  matrices over the ring â„¤ / p^n for p prime and n positive.
  
  If a ring R is supported in [5XMatricesForHomalg[0m any of its residue class rings
  R/I  is supported as well, provided the ideal I of relations admits a finite
  set  of  generators  as  a  left resp. right ideal (--> [2X\/[0m ([14X3.2-3[0m)). This is
  immediate for commutative noetherian rings.
  
  [1X3.2-1 HomalgRingOfIntegers[0m
  
  [2X> HomalgRingOfIntegers( [0X[3X[0X[2X ) _________________________________________[0Xfunction
  [6XReturns:[0X  a [5Xhomalg[0m ring
  
  [2X> HomalgRingOfIntegers( [0X[3Xc[0X[2X ) ________________________________________[0Xfunction
  [6XReturns:[0X  a [5Xhomalg[0m ring
  
  The no-argument form returns the ring of integers â„¤ for [5Xhomalg[0m.
  
  The  one-argument  form  accepts an integer [3Xc[0m and returns the ring â„¤ / c for
  [5Xhomalg[0m:
  
  --    [3Xc[0m= 0 defaults to â„¤
  
  --    if [3Xc[0m is a prime power then the package [5XGaussForHomalg[0m is loaded (if it
        fails to load an error is issued)
  
  --    otherwise,  the  residue  class ring constructor [10X/[0m (--> [2X\/[0m ([14X3.2-3[0m)) is
        invoked
  
  The  operation  [10XSetRingProperties[0m  is  automatically invoked to set the ring
  properties.
  
  If  for  some reason you don't want to use the [5XGaussForHomalg[0m package (maybe
  because you didn't install it), then use
  
  [10XHomalgRingOfIntegers[0m( ) [10X/[0m [3Xc[0m;
  
  but note that the computations will then be considerably slower.
  
  [1X3.2-2 HomalgFieldOfRationals[0m
  
  [2X> HomalgFieldOfRationals( [0X[3X[0X[2X ) _______________________________________[0Xfunction
  [6XReturns:[0X  a [5Xhomalg[0m ring
  
  The  package  [5XGaussForHomalg[0m  is  loaded  and  the  field  of rationals â„š is
  returned. If [5XGaussForHomalg[0m fails to load an error is issued.
  
  The  operation  [10XSetRingProperties[0m  is  automatically invoked to set the ring

  [1X3.2-3 \/[0m
  
  [2X> \/( [0X[3XR, ring_rel[0X[2X ) _______________________________________________[0Xoperation
  [6XReturns:[0X  a [5Xhomalg[0m ring
  
  This  is  the [5Xhomalg[0m constructor for residue class rings [3XR[0m / I, where [3XR[0m is a
  [5Xhomalg[0m  ring and I=[3Xring_rel[0m is the ideal of relations generated by [3Xring_rel[0m.
  [3Xring_rel[0m might be:
  
  --    a set of ring relations of a left resp. right ideal
  
  --    a list of ring elements of [3XR[0m
  
  --    a ring element of [3XR[0m
  
  For noncommutative rings: In the first case the set of ring relations should
  generate  the  ideal  of  relations  I  as left resp. right ideal, and their
  involutions  should generate I as right resp. left ideal. If [3Xring_rel[0m is not
  a set of relations, a [13Xleft[0m set of relations is constructed.
  
  The  operation  [10XSetRingProperties[0m  is  automatically invoked to set the ring
  properties.

  [1X3.3 Rings: Properties[0X
  
  The  following  properties  are  declared for [5Xhomalg[0m rings. Note that (apart
  from  so-called  true  and immediate methods (--> [14XC.1[0m)) there are no methods
  installed  for  ring  properties.  This  means that if the value of the ring
  property [10XProp[0m is not set for a [5Xhomalg[0m ring [3XR[0m, then
  
  [10XProp[0m( [3XR[0m );
  
  will  cause  an  error. One can use the usual [5XGAP4[0m mechanism to check if the
  value of the property is set or not
  
  [10XHasProp[0m( [3XR[0m );
  
  If  you  discover  that  a  specific  property [10XProp[0m is missing for a certain
  [5Xhomalg[0m ring [3XR[0m you can it add using the usual [5XGAP4[0m mechanism
  
  [10XSetProp[0m( [3XR[0m, true );
  
  or
  
  [10XSetProp[0m( [3XR[0m, false );
  
  Be very cautious with setting "missing" properties to [5Xhomalg[0m objects: If the
  value  you  set  is  mathematically  wrong  [5Xhomalg[0m  will probably draw wrong
  conclusions and might return wrong results.
