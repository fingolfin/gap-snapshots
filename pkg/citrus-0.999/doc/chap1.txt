  
  [1X1 [33X[0;0YThe [5XCitrus[105X[101X[1X package[133X[101X
  
  [33X[0;0YThis  is  the manual for the [5XCitrus[105X package version 0.999 for computing with
  semigroups  of  transformations and partial permutations. [5XCitrus[105X 0.999 is an
  updated   and   expanded   version   of   the   Monoid  package  for  GAP  3
  ([7Xhttp://schmidt.nuigalway.ie/monoid/index.html[107X)  by Goetz Pfeiffer, Steve A.
  Linton, Edmund F. Robertson, and Nik Ruskuc and the Monoid package for GAP 4
  ([7Xhttp://www-history.mcs.st-and.ac.uk/~jamesm/monoid/index.html[107X)   by  J.  D.
  Mitchell.[133X
  
  [33X[0;0YSome  of the theory behind the algorithms in [5XCitrus[105X is described in [LPRR98]
  and described in [LPRR02]. Another reference is [LM90].[133X
  
  [33X[0;0YThe  functionality  of [5XMonoid[105X 3.1.4 for [5XGAP[105X 4 has been split between the [5XGAP[105X
  4.5 packages [5XCitrus[105X and [5XComatose[105X. [5XCitrus[105X 0.999 retains all the functionality
  of  the  original [5XMonoid[105X package for [5XGAP[105X 3; and many of those functions from
  [5XMonoid[105X  3.1.4  not  involved  in  the  computation of automorphism groups of
  semigroups.  The  [5XComatose[105X package retains those functions from [5XMonoid[105X 3.1.4
  used to compute automorphism groups of transformation semigroups.[133X
  
  [33X[0;0Y[5XCitrus[105X 0.999 contains more efficient methods than those available in the [5XGAP[105X
  library  (and  in  many  cases  more  efficient than any other software) for
  creating semigroups of transformations and partial permutations, calculating
  their  Green's  classes,  size, elements, group of units, minimal ideal, and
  testing   membership,  finding  the  inverses  of  a  regular  element,  and
  factorizing  elements over the generators, and many more; see Chapters [14X3[114X and
  [14X4[114X.  There are also methods for testing if a semigroup satisfies a particular
  property, such as if it is regular, simple, inverse, completely regular, and
  a  variety  of  further  properties;  see Chapter [14X4[114X. A range of functions is
  provided   for   creating   and   determining   properties   of   individual
  transformations and partial permutations such as the index and period or the
  least  idempotent  power;  see  Chapter  [14X2[114X. A large catalogue of examples is
  provided;  see  Sections [14X3.4[114X and [14X3.5[114X. [5XCitrus[105X also provides abbreviated names
  for  many  of the commonly used [5XGAP[105X library functions related to semigroups,
  and  functions  to  read  and  write large collections of transformations or
  partial  permutations  to  a  file;  see  [2XReadCitrus[102X ([14X1.6-2[114X) and [2XWriteCitrus[102X
  ([14X1.6-3[114X).[133X
  
  [33X[0;0YThe  [5XCitrus[105X package is written in C and [5XGAP[105X code and requires the [5XOrb[105X and [5XIO[105X
  packages.  The  [5XOrb[105X  package  is  used  to  efficiently  compute  orbits  in
  semigroups  of  transformations  and partial permutations, and these methods
  underpin  many of the features of [5XCitrus[105X. The [5XIO[105X package is used to read and
  write  transformations and partial permutations to a file. The [5XGrape[105X package
  is used in the function [2XMunnSemigroup[102X ([14X3.4-3[114X) but nowhere else in [5XCitrus[105X. If
  [5XGrape[105X  is  not  fully  installed, then [5XCitrus[105X can be used as normal with the
  except that the operation [2XMunnSemigroup[102X ([14X3.4-3[114X) does not work.[133X
  
  [33X[0;0Y[5XCitrus[105X  is still under development, and so some features may not function as
  expected. At the present time, I do not know of any errors or serious issues
  with [5XCitrus[105X. If you find a bug or an issue with the package, then please let
  me know by emailing me the details at: [7Xmailto:jdm3@st-and.ac.uk[107X.[133X
  
  [33X[0;0YFor more details about semigroups in [5XGAP[105X or Green's relations in particular,
  see [14X'Reference: Semigroups'[114X or [14X'Reference: Green's Relations'[114X.[133X
  
  
  [1X1.1 [33X[0;0YInstalling [5XCitrus[105X[101X[1X[133X[101X

        details go to:[133X

        io.html[107X[133X

  [1X1.2 [33X[0;0YCompiling the kernel component of [5XCitrus[105X[101X[1X[133X[101X
  
  [33X[0;0YAs  of version 0.7, the [5XCitrus[105X package has a [5XGAP[105X kernel component in C which
  should  be compiled. This component contains low-level functions relating to
  partial  permutations  and  it  is not possible to use these features or any
  related  features in [5XCitrus[105X without compiling the package. It is possible to
  run  [5XCitrus[105X  without  compiling  it, but [5XCitrus[105X is limited to computing with
  transformation semigroups in this case.[133X
  
  [33X[0;0YTo compile the GAP kernel component in the 'pkg' directory do:[133X
  
  [4X[32X[104X
    [4Xcd citrus[104X
    [4X./configure[104X
    [4Xmake[104X
  [4X[32X[104X
  
  [33X[0;0YIf  you  installed  the package in another 'pkg' directory than the standard
  'pkg'  directory  in  your [5XGAP[105X installation, then you have to do two things.
  Firstly  during compilation you have to use the option '--with-gaproot=PATH'
  of  the  'configure'  script  where  'PATH'  is  a path to the main GAP root
  directory (if not given the default '../..' is assumed).[133X
  
  [33X[0;0YIf  you  installed  [5XGAP[105X  on  several  architectures,  you  must  execute the
  configure/make  step  for  each of the architectures. You can either do this
  immediately after configuring and compiling GAP itself on this architecture,
  or  alternatively  (when  using  version  4.5  of  GAP  or  newer)  set  the
  environment  variable 'CONFIGNAME' to the name of the configuration you used
  when  compiling  GAP  before  running  './configure'. Note however that your
  compiler  choice and flags (environment variables 'CC' and 'CFLAGS') need to
  be  chosen  to  match the setup of the original GAP compilation. For example
  you have to specify 32-bit or 64-bit mode correctly![133X

  [1X1.6-3 WriteCitrus[101X
  
  [29X[2XWriteCitrus[102X( [3Xfilename[103X, [3Xlist[103X ) [32X function
  [6XReturns:[106X  [33X[0;10YNothing.[133X
  
  [33X[0;0YThis  function  provides  a  method  for writing transformations and partial
  permutations  to  a  file that uses a relatively small amount of disk space.
  The resulting file can be further compressed using [10Xgzip[110X.[133X
  
  [33X[0;0Y[3Xlist[103X  should  be  a  list  of:  transformations  of  equal  degree,  partial
  permutations,  a  semigroup of transformations or partial permutations, or a
  list  of  lists  of  transformations, partial permutations, or semigroups of
  transformations  or  partial  permutations;  and [3Xfilename[103X should be a string
  containing the name of a file where the entries in [3Xlist[103X will be written.[133X
  
  [33X[0;0Y[10XWriteCitrus[110X  appends a line to the file [3Xfilename[103X for every entry in [3Xlist[103X. If
  any  element  of  [3Xlist[103X is a semigroup, then the generators of that semigroup
  are written to [3Xfilename[103X.[133X
  
  [33X[0;0YThe  first  character  [10Xm[110X of the appended line is the number of characters in
  the  degree  of the transformations to be written, the next [10Xm[110X characters are
  the  degree  [10Xn[110X of the transformations to be written, and the transformations
  themselves  are  written  in blocks of [10Xm*n[110X in the remainder of the line. For
  example, the transformations:[133X
  
  [4X[32X  Example  [32X[104X
    [4X[28X[ Transformation( [ 2, 6, 7, 2, 6, 9, 9, 1, 1, 5 ] ), [128X[104X
    [4X[28X  Transformation( [ 3, 8, 1, 9, 9, 4, 10, 5, 10, 6 ] )][128X[104X
  [4X[32X[104X
  
  [33X[0;0Yare written as:[133X
  
  [4X[32X  Example  [32X[104X
    [4X[28X210 2 2 6 7 2 6 9 9 1 1 5 3 8 1 9 9 410 510 6[128X[104X
  [4X[32X[104X
  
  [33X[0;0YThe file [3Xfilename[103X can be read using [2XReadCitrus[102X ([14X1.6-2[114X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xfile:=Concatenation(CitrusDir(), "/examples/graph7c.citrus.gz");;[127X[104X

    [4X[28X[ Transformation( [ 1, 2, 2, 4, 5, 6, 7 ] ), [128X[104X
    [4X[28X  Transformation( [ 1, 2, 3, 4, 5, 6, 7 ] ), [128X[104X
    [4X[28X  Transformation( [ 1, 2, 3, 4, 5, 7, 7 ] ), [128X[104X
    [4X[28X  Transformation( [ 1, 3, 2, 4, 7, 6, 7 ] ), [128X[104X
    [4X[28X  Transformation( [ 4, 2, 1, 1, 6, 5, 7 ] ), [128X[104X
    [4X[28X  Transformation( [ 4, 3, 2, 1, 6, 7, 7 ] ), [128X[104X
    [4X[28X  Transformation( [ 4, 4, 5, 7, 6, 1, 1 ] ), [128X[104X
    [4X[28X  Transformation( [ 7, 6, 6, 1, 2, 4, 4 ] ), [128X[104X

  [1X1.7 [33X[0;0YWhat does [5XCitrus[105X[101X[1X do best?[133X[101X
  
  [33X[0;0YDue to inherent difficulties with computing Green's [13XL[113X-, [13XD[113X-, and [13XH[113X-classes in
  transformation   semigroups,  the  methods  used  to  compute  with  Green's
  [13XR[113X-classes  are the most efficient in [5XCitrus[105X. Thus, if you are computing with
  a  transformation  semigroup,  wherever  possible it is advisable to use the
  commands relating to Green's [13XR[113X-classes rather than those relating to Green's
  [13XL[113X-,  [13XD[113X-,  or [13XH[113X-classes. No such difficulties are present when computing with
  semigroups of partial permutations.[133X
  
  [33X[0;0YThe  methods  in  [5XCitrus[105X allow the computation of individual Green's classes
  without  the  need  to compute all the elements of the underlying semigroup;
