  [1X4.1 Methods and functions for orbifold triangulations[0X
  
  [1X4.1-1 OrbifoldTriangulation[0m
  
  [2X> OrbifoldTriangulation( [0X[3XM[, I, mu_data, info][0X[2X ) ___________________[0Xfunction
  [6XReturns:[0X  OrbifoldTriangulation
  
  The  constructor  for  OrbifoldTriangulations.  Needs  the list [3XM[0m of maximal
  simplices,  the  Isotropy  at  certain  vertices as a record [3XI[0m, and the list
  [3Xmu_data[0m  that  encodes the function mu. If only one argument is given, [3XI[0m and
  [3Xmu_data[0m  are  supposed  to  be  empty.  In case of two arguments, [3Xmu_data[0m is
  supposed  to be empty. If the last argument [3Xinfo[0m is given as a string, it is
  stored  in  the  info  component  of the orbifold triangulation and does not
  count towards the total number of arguments.
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> M := [ [1,2,3], [1,2,4], [1,3,4], [2,3,4] ];;[0X
    [4Xgap> S2 := OrbifoldTriangulation( M, "S^2" );[0X
    [4X<OrbifoldTriangulation "S^2" of dimension 2. 4 simplices on 4 vertices without\[0X
    [4X Isotropy>[0X
    [4Xgap> I := rec( 1 := Group( (1,2) ) );;[0X
    [4Xgap> mu_data := [[0X
    [4X> [ [2], [1,2], [1,2,3], [1,2,4], x->x*(1,2) ],[0X
    [4X> [ [2], [1,2], [1,2,4], [1,2,3], x->x*(1,2) ][0X
    [4X> ];;[0X
    [4Xgap> Teardrop := OrbifoldTriangulation( M, I, mu_data, "Teardrop" );[0X
    [4X<OrbifoldTriangulation "Teardrop" of dimension 2. 4 simplices on 4 vertices wi\[0X
    [4Xth Isotropy on 1 vertex and nontrivial mu-maps>[0X
  [4X------------------------------------------------------------------[0X

  [1X4.2 Methods and functions for simplicial sets[0X
  
  [1X4.2-1 SimplicialSet[0m
  
  [2X> SimplicialSet( [0X[3Xot[0X[2X ) ________________________________________________[0Xmethod
  [6XReturns:[0X  SimplicialSet
  
  The  constructor  for simplicial sets based on an orbifold triangulation [3Xot[0m.
  This  just  sets  up  the  object  without  any  computations.  These can be
  triggered later, either explicitly or by [2XSimplicialSet[0m ([14X4.2-2[0m).
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> Teardrop;[0X
    [4X<OrbifoldTriangulation "Teardrop" of dimension 2. 4 simplices on 4 vertices wi\[0X
    [4Xth Isotropy on 1 vertex and nontrivial mu-maps>[0X
    [4Xgap> S := SimplicialSet( Teardrop );[0X
    [4X<The simplicial set of the orbifold triangulation "Teardrop", computed up to d\[0X
    [4Ximension 0 with Length vector [ 4 ]>[0X
  [4X------------------------------------------------------------------[0X
  
  [1X4.2-2 SimplicialSet[0m
  
  [2X> SimplicialSet( [0X[3XS, i[0X[2X ) ______________________________________________[0Xmethod
  [6XReturns:[0X  List [3XS[0m_[3Xi[0m
  
  This  returns  the components of dimension [3Xi[0m of the simplicial set [3XS[0m. Should

  [1X4.3 Methods and functions for matrix creation and computation[0X
  
  [1X4.3-1 BoundaryOperator[0m
  
  [2X> BoundaryOperator( [0X[3Xi, L, mu[0X[2X ) _____________________________________[0Xfunction
  [6XReturns:[0X  List B
  
  This  returns  the  [3Xi[0mth  boundary  of  [3XL[0m,  which  has  to be an element of a
  simplicial set. [3Xmu[0m is the function mu that has to be taken into account when
  computing  orbifold  boundaries.  This function is used for matrix creation,
  there should not be much reason for calling it independently.
  
  [1X4.3-2 CreateBoundaryMatrices[0m
  
  [2X> CreateBoundaryMatrices( [0X[3XS, d, R[0X[2X ) __________________________________[0Xmethod
  [6XReturns:[0X  List [3XM[0m
  
  This  returns  the  list  [3XM[0m  of homalg matrices over the homalg ring [3XR[0m up to
  dimension   [3Xd[0m,  corresponding  to  the  boundary  matrices  induced  by  the
  simplicial set [3XS[0m. If [3Xd[0m is not given, the current dimension of [3XS[0m is used.
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> S := SimplicialSet( Teardrop );[0X
    [4X<The simplicial set of the orbifold triangulation "Teardrop", computed up to d\[0X
    [4Ximension 0 with Length vector [ 4 ]>[0X
    [4Xgap> M := CreateBoundaryMatrices( S, 4, HomalgRingOfIntegers() );;[0X
    [4Xgap> S;[0X
    [4X<The simplicial set of the orbifold triangulation "Teardrop", computed up to d\[0X
    [4Ximension 5 with Length vector [ 4, 12, 22, 33, 51, 73 ]>[0X
  [4X------------------------------------------------------------------[0X
