  
  [1X3 Examples[0X
  
  Although  there  are some small examples embedded in chapter [14X4[0m, we will give
  some complete examples in this chapter. Most of these could easily be called
  with  the example script mentioned in chapter [14X2[0m, but we will do them step by
  step for best reproducability.
  
  
  [1X3.1 Example 1: Klein Bottle[0X
  
  Suppose  we  want to calculate the cohomology of the Klein Bottle. First, we
  need a triangulation. It could look like this:
  
  [4X------------------------------------------------------------------[0X
    [4X1--2--3--1[0X
    [4X|\ |\ |\ |[0X
    [4X| \| \| \|[0X
    [4X4--5--6--4[0X
    [4X|\ |\ |\ |[0X
    [4X| \| \| \|[0X
    [4X7--8--9--7[0X
    [4X|\ |\ |\ |[0X
    [4X| \| \| \|[0X
    [4X1--3--2--1[0X
  [4X------------------------------------------------------------------[0X
  
  This results in the following list of maximum simplices:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> M := [ [1,2,4], [1,2,7], [1,3,6], [1,3,8], [1,4,6], [1,7,8],[0X
    [4X> [2,3,5], [2,3,9], [2,4,5], [2,7,9], [3,5,6], [3,8,9],[0X
    [4X> [4,5,7], [4,6,9], [4,7,9], [5,6,8], [5,7,8], [6,8,9] ];;[0X
  [4X------------------------------------------------------------------[0X
  
  As  there  is  no isotropy and therefore no mu-map, we can continue with the
  orbifold triangulation and simplicial set:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> ot := OrbifoldTriangulation( M, "Klein Bottle" );[0X
    [4X<OrbifoldTriangulation "Klein Bottle" of dimension 2. 18 simplices on 9 vertic\[0X
    [4Xes without Isotropy>[0X
    [4Xgap> ss := SimplicialSet( ot );[0X
    [4X<The simplicial set of the orbifold triangulation "Klein Bottle", computed up \[0X
    [4Xto dimension 0 with Length vector [ 18 ]>[0X
  [4X------------------------------------------------------------------[0X
  
  Now  we  will  need a [5Xhomalg[0m ring. As this is a small example we can compute
  directly  over  â„¤,  so  we  can  use  [5XGAP[0m.  In  case you have [5XRingsForHomalg[0m
  installed you might want to try computing in another computer algebra system
  with  the  command  [10XHomalgRingOfIntegersInCAS()[0m,  replacing  "CAS"  with the
  corresponding system.
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> R := HomalgRingOfIntegers();[0X
    [4XZ[0X
  [4X------------------------------------------------------------------[0X
  
  We  are  almost  there.  Let  us create some coboundary matrices and compute
  their cohomology:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> c := CreateCoboundaryMatrices( ss, 4, R );;[0X
    [4Xgap> C := Cohomology( c, R );[0X
    [4X----------------------------------------------->>>>  Z^(1 x 1)[0X
    [4X----------------------------------------------->>>>  Z^(1 x 1)[0X
    [4X----------------------------------------------->>>>  Z/< 2 >[0X
    [4X----------------------------------------------->>>>  0[0X
    [4X----------------------------------------------->>>>  0[0X
    [4X<A graded cohomology object consisting of 5 left modules at degrees[0X
    [4X[ 0 .. 4 ]>[0X
  [4X------------------------------------------------------------------[0X
  
  This is the cohomology of the Klein Bottle.
  
  
  [1X3.2 Example 2: V_4[0X
  
  [5XSCO[0m  can  also  be  used  to compute group cohomology, as every group can be
  represented  as an orbifold with just a single point. For V_4, it would look
  like this:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> M := [ [1] ];;[0X
    [4Xgap> V4 := Group( (1,2), (3,4) );;[0X
    [4Xgap> iso := rec( 1 := V4 );;[0X
    [4Xgap> ot := OrbifoldTriangulation( M, iso, "V4" );[0X
    [4X<OrbifoldTriangulation "V4" of dimension 0. 1 simplex on 1 vertex with Isotrop\[0X
    [4Xy on 1 vertex>[0X
    [4Xgap> ss := SimplicialSet( ot );[0X
    [4X<The simplicial set of the orbifold triangulation "V4", computed up to dimensi\[0X
    [4Xon 0 with Length vector [ 1 ]>[0X
    [4Xgap> R := HomalgRingOfIntegers();[0X
    [4XZ[0X
    [4Xgap> c := CreateCoboundaryMatrices( ss, 4, R );;[0X
    [4Xgap> C := Cohomology( c, R );[0X
    [4X----------------------------------------------->>>>  Z^(1 x 1)[0X
    [4X----------------------------------------------->>>>  0[0X
    [4X----------------------------------------------->>>>  Z/< 2 > + Z/< 2 >[0X
    [4X----------------------------------------------->>>>  Z/< 2 >[0X
    [4X----------------------------------------------->>>>  Z/< 2 > + Z/< 2 > + Z/< 2\[0X
    [4X >[0X
    [4X<A graded cohomology object consisting of 5 left modules at degrees[0X
    [4X[ 0 .. 4 ]>[0X
  [4X------------------------------------------------------------------[0X
  
  This is the V_4 group cohomology up to degree 4.
  
  
  [1X3.3 Example 3: The "Teardrop" orbifold[0X
  
  You  have  seen  a manifold in example 1, and group cohomology in example 2.
  Now  we  will  meet  our  first  proper  orbifold, the Teardrop. This is the
  example  Moerdijk  and  Pronk used in their paper [MP99] on which my work is
  based.  It  is  an  easy  example, but includes both nontrivial isotropy and
  mu-maps.  We take the isotropy at the top to be C_2. The triangulation looks
  like this, with the gluing being at [1,3].
  
  [4X------------------------------------------------------------------[0X
    [4X   3=====1=====3[0X
    [4X  / \   / \   / \[0X
    [4X /   \ /   \ /   \[0X
    [4X5-----2-----4-----5[0X
    [4X       \   /[0X
    [4X        \ /[0X
    [4X         5[0X
  [4X------------------------------------------------------------------[0X
  
  The source code:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> M := [ [1,2,3], [1,2,4], [1,3,4], [2,3,5], [2,4,5], [3,4,5] ];;[0X
    [4Xgap> iso := rec( 1 := Group( (1,2) ) );;[0X
    [4Xgap> mu := [[0X
    [4X>            [ [3], [1,3], [1,2,3], [1,3,4], x -> (1,2) ],[0X
    [4X>            [ [3], [1,3], [1,3,4], [1,2,3], x -> (1,2) ][0X
    [4X>          ];;[0X
    [4Xgap> ot := OrbifoldTriangulation( M, iso, mu, "Teardrop" );[0X
    [4X<OrbifoldTriangulation "Teardrop" of dimension 2. 6 simplices on 5 vertices wi\[0X
    [4Xth Isotropy on 1 vertex and nontrivial mu-maps>[0X
    [4Xgap> ss := SimplicialSet( ot );[0X
    [4X<The simplicial set of the orbifold triangulation "Teardrop", computed up to d\[0X
    [4Ximension 0 with Length vector [ 6 ]>[0X
    [4Xgap> R := HomalgRingOfIntegers();[0X
    [4XZ[0X
    [4Xgap> c := CreateCoboundaryMatrices( ss, 6, R );;[0X
    [4Xgap> C := Cohomology( c, R );[0X
    [4X----------------------------------------------->>>>  Z^(1 x 1)[0X
    [4X----------------------------------------------->>>>  0[0X
    [4X----------------------------------------------->>>>  Z^(1 x 1)[0X
    [4X----------------------------------------------->>>>  0[0X
    [4X----------------------------------------------->>>>  Z/< 2 >[0X
    [4X----------------------------------------------->>>>  0[0X
    [4X----------------------------------------------->>>>  Z/< 2 >[0X
    [4X<A graded cohomology object consisting of 7 left modules at degrees[0X
    [4X[ 0 .. 6 ]>[0X
  [4X------------------------------------------------------------------[0X
  
  This is the Teardrop cohomology.
