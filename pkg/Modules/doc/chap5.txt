  
  [1X5 Relations[0X
  
  A finite presentation of a module is given by a finite set of generators and
  a  finite  set  of  relations  among  these  generators.  In [5Xhomalg[0m a set of
  relations  of a left/right module is given by a matrix [3Xrel[0m, the rows/columns
  of which are interpreted as relations among n generators, n being the number
  of columns/rows of the matrix [3Xrel[0m.
  
  The data structure of a module in [5Xhomalg[0m is designed to contain not only one
  but   several  sets  of  relations  (together  with  corresponding  sets  of
  generators (--> Chapter [14X6[0m)). The different sets of relations are linked with
  so-called transition matrices (--> Chapter [14X7[0m).
  
  The  relations  of  a [5Xhomalg[0m module are evaluated in a lazy way. This avoids
  unnecessary computations.
