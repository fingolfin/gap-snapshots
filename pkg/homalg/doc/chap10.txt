  
  [1X10 Spectral Sequences[0X
  
  Spectral  sequences  are  regarded  as  the  computational  sledgehammer  in
  homological algebra. Quoting the last lines of Rotman's book [Rot79]:
  
  "The  reader should now be convinced that virtually every purely homological
  result  may  be  proved  with  spectral  sequences. Even though "elementary"
  proofs  may  exist  for  many  of  these results, spectral sequences offer a
  systematic approach in place of sporadic success."

  [1X10.2 Spectral Sequences: Constructors[0X
  
  [1X10.2-1 HomalgSpectralSequence[0m
  
  [2X> HomalgSpectralSequence( [0X[3Xr, B, a[0X[2X ) _______________________________[0Xoperation
  [2X> HomalgSpectralSequence( [0X[3Xr, B[0X[2X ) __________________________________[0Xoperation
  [2X> HomalgSpectralSequence( [0X[3XB, a[0X[2X ) __________________________________[0Xoperation
  [2X> HomalgSpectralSequence( [0X[3XB[0X[2X ) _____________________________________[0Xoperation
  [6XReturns:[0X  a [5Xhomalg[0m spectral sequence
  
  The  first syntax is the main constructor. It creates the homological (resp.
  cohomological)  spectral  sequence  associated  to  the  homological  (resp.
  cohomological)  bicomplex  [3XB[0m  starting  at level 0 and ending at level [3Xr[0m>= 0
  (regardless  if  the  spectral sequence stabilizes earlier). The generalized
  embeddings  into  the  objects  of  0-th  sheet are always computed for each
  higher   sheet   Er   and   stored   as   a   record   under  the  component
  Er!.absolute_embeddings.  If  [3Xa[0m is greater than 0 the generalized embeddings
  into  the  objects of the [3Xa[0m-th sheet also get computed for each higher sheet
  Er  and  stored as a record under the component Er!.relative_embeddings. The
  level  [3Xa[0m  at  which  the  spectral  sequence  becomes intrinsic is a natural
  candidate for [3Xa[0m. The [3Xa[0m-th sheet is called the [13Xspecial[0m sheet.
  
  If  [3Xr[0m=-1  it  computes  all  the  sheets  of the spectral sequence until the
  sequence stabilizes, i.e. until all higher arrows become zero.
  
  If [3Xa[0m=-1 no special sheet is specified.
  
  In the second syntax [3Xa[0m is set to -1.
  
  In the third syntax [3Xr[0m is set to -1.
  
  In the fourth syntax both [3Xr[0m and [3Xa[0m are set to -1.
  
  The  following  example  demonstrates  the computation of a Tor-Ext spectral
  sequence:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> ZZ := HomalgRingOfIntegers( );[0X
    [4XZ[0X
    [4Xgap> M := HomalgMatrix( "[ 2, 3, 4,   5, 6, 7 ]", 2, 3, ZZ );;[0X
    [4Xgap> M := LeftPresentation( M );[0X
    [4X<A non-torsion left module presented by 2 relations for 3 generators>[0X
    [4Xgap> dM := Resolution( M );[0X
    [4X<A non-zero right acyclic complex containing a single morphism of left modules\[0X
    [4X at degrees [ 0 .. 1 ]>[0X
    [4Xgap> CC := Hom( dM, dM );[0X
    [4X<A non-zero acyclic cocomplex containing a single morphism of right complexes \[0X
    [4Xat degrees [ 0 .. 1 ]>[0X
    [4Xgap> B := HomalgBicomplex( CC );[0X
    [4X<A non-zero bicocomplex containing right modules at bidegrees [ 0 .. 1 ]x[0X
    [4X[ -1 .. 0 ]>[0X
  [4X------------------------------------------------------------------[0X
  
  Now  we  construct the spectral sequence associated to the bicomplex B, also
  called the [13Xfirst[0m spectral sequence:
  
  [4X---------------------------  Example  ----------------------------[0X
    [4Xgap> I_E := HomalgSpectralSequence( 2, B );[0X
    [4X<A stable cohomological spectral sequence with sheets at levels [0X
    [4X[ 0 .. 2 ] each consisting of right modules at bidegrees [ 0 .. 1 ]x[0X
    [4X[ -1 .. 0 ]>[0X
    [4Xgap> Display( I_E );[0X
    [4Xa cohomological spectral sequence at bidegrees[0X
    [4X[ [ 0 .. 1 ], [ -1 .. 0 ] ][0X
    [4X---------[0X
    [4XLevel 0:[0X
    [4X[0X
    [4X * *[0X
    [4X * *[0X
    [4X---------[0X
    [4XLevel 1:[0X
    [4X[0X
    [4X * *[0X
    [4X . .[0X
    [4X---------[0X
    [4XLevel 2:[0X
    [4X[0X
    [4X s s[0X
    [4X . .[0X
  [4X------------------------------------------------------------------[0X
  
  Legend:
  
  --    A star [3X*[0m stands for a nonzero object.
  
  --    A dot [3X.[0m stands for a zero object.
  
  --    The letter [3Xs[0m stands for a nonzero object that became stable.
  
  The  [13Xsecond[0m  spectral  sequence  of  the  bicomplex  is,  by definition, the
  spectral sequence associated to the transposed bicomplex:

    [4Xgap> tB := TransposedBicomplex( B );[0X
