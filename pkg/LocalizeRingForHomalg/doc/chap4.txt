  
  [1X4 Localize Rings[0X
  
  The  package [5XLocalizeRingForHomalg[0m defines the classes of local(ized) rings,
  local  ring  elements and local matrices. These three objects can be used as
  data structures defined in [5XMatricesForHomalg[0m on which the [5Xhomalg[0m project can
  rely to do homological computations over localized rings.
  
  A  [5Xhomalg[0m  local ring element contains two [5Xhomalg[0m ring elements, a numerator
  (-->  [2XNumerator[0m  ([14X4.3-4[0m))  and  a  denominator  (--> [2XDenominator[0m ([14X4.3-6[0m)). A
  [5Xhomalg[0m  local  matrix  contains  a  global [5Xhomalg[0m matrix as a numerator (-->
  [2XNumerator[0m  ([14X4.3-5[0m))  and  a  ring  element as a denominator (--> [2XDenominator[0m
  ([14X4.3-7[0m)).   New   constructors   for   ring   elements   and   matrices  are
  [2XHomalgLocalRingElement[0m  ([14X4.3-17[0m)  and [2XHomalgLocalMatrix[0m ([14X4.3-18[0m) in addition
  to  the  standard  contructors  introduced  in  other packages of the [5Xhomalg[0m
  project.
  
  The local rings most prominently can be used with methods known from general
  [5Xhomalg[0m  rings.  The  methods for doing the computations are presented in the
  appendix  ([14XA[0m),  since  they are not for external use. The new attributes and
  operations are documented here.
  
  Since   the  objects  inplemented  here  are  representations  from  objects
  elsewhere  in the [5Xhomalg[0m project (i.e. [5XMatricesForHomalg[0m), we want to stress
  that  there  are  many  other  operations  in  [5Xhomalg[0m,  which can be used in
  connection  with  the ones presented here. A few of them can be found in the
  examples and the appendix of this documentation.

  [1X4.3-13 Cancel[0m
  
  [2X> Cancel( [0X[3Xa, b[0X[2X ) __________________________________________________[0Xoperation
  [6XReturns:[0X  two ring elements
  
  For  [3Xa[0m=a'*c  and  [3Xb[0m=b'*c  return  a'  and b'. The exact form of c depends on
  whether a procedure for gcd computation is included in the ring package.
  
  [1X4.3-14 LocalizeAt[0m
  
  [2X> LocalizeAt( [0X[3XR, l[0X[2X ) ______________________________________________[0Xoperation
  [6XReturns:[0X  a local ring
  
  If  [3Xl[0m is a list of elements of the global ring [3XR[0m generating a maximal ideal,
  the  method creates the corresponding localization of [3XR[0m at the complement of
  the maximal ideal.
  
  [1X4.3-15 LocalizeAtZero[0m
  
  [2X> LocalizeAtZero( [0X[3XR[0X[2X ) _____________________________________________[0Xoperation
  [6XReturns:[0X  a local ring
  
  This method creates the corresponding localization of [3XR[0m at the complement of
  the maximal ideal generated by the indeterminates ("at zero").
  
  [1X4.3-16 LocalizePolynomialRingAtZeroWithMora[0m
  
  [2X> LocalizePolynomialRingAtZeroWithMora( [0X[3XR[0X[2X ) _______________________[0Xoperation
  [6XReturns:[0X  a local ring
  
  This  method  localizes  the  ring  [3XR[0m  at  zero  and  this localized ring is
  returned.  The  ring table uses Mora's algorithm as implemented [5XSingular[0m for
  low level computations.
